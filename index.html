mkflickr

mkflickr is a jquery plugin to build a pretty gallery of images based on a users Flickr userID. Please note, you need a Flickr developers API key to use this plugin.

Demo

This is a VERY early demo, as the plugin is still very much WIP, but at least it shows images from Flickr ;-).

Documentation

Using the plugin is pretty simple. Just follow these steps...

Required HTML markup

The plugin requires a target DIV element in your page (only tested with DIVs so far, but this may well change).

<div id="gallery-region"></div>

Include the jQuery Library

If your project is not using jQuery already, make sure you include it. the mkFlickr plugin repo includes a copy of jquery-1.10.1.js.

<script src="build/js/query-1.10.1.min.js"></script>

Include the mkflickr plugin

While you can create your own CSS for the gallery or omit CSS entirely if you really wanted to, you must include the jquery.mkflickr.js file in your page

<script src="build/js/jquery.mkflickr.min.js"></script>

Initialising the plugin

Finally, initialise the plugin

$(document).ready(function () {
  $("#gallery-test").mkflickr({
    apiKey: 'YOUR FLICKR API KEY',
    userId: 'YOUR USER ID',
    itemCount: 10,
    tag: 'span'
  });
});

Plugin Options

  • apiKey: Your Flickr developers API key as a string, eg: 'd980cbf534dffb23fa2611329cc10f98'
  • itemCount: Your Flickr user's id as a string, eg: '44123892@N01'
  • userID: Number of images you would like returned as a number (default is 20)
  • tag: The html tag you would like to wrap each image in (eg: 'span', 'div'). Coming soon, more flexibility and markup options.
